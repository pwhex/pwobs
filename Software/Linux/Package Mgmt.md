Archlinux - Pacman

Installation
standard syntax: pacman -S <package1> <packaage2>
-S stands for synchronization. This is what users use for installation of files.

installed files are two fold:
1. explicit install files
2. dependencies

Removal
standard syntax: pacman -R <packagename>

remove a package and its dependencies: pacman -Rs
